# 快速编辑自定义名称

## 快捷键说明

### 基本操作

- **双击单元格**：进入编辑模式
- **Enter 键**：保存当前编辑并跳转到下一行的自定义名称列继续编辑
- **ESC 键**：取消本次编辑，忽略所有修改
- **失去焦点**：自动保存当前编辑

### 批量操作

- **框选多行**：支持鼠标框选或按住 Ctrl/Shift 键点击多行进行选择
- **Ctrl+A**：全选所有行

- **Ctrl+V**：粘贴多行文本（Excel 风格）
  - 框选或选中多行后，按 Ctrl+V 粘贴
  - 第一行文本粘贴到第一个选中项，第二行粘贴到第二个选中项，以此类推
  - 支持从 Excel、记事本等应用复制多行文本
  - 如果粘贴的行数少于选中项数，只填充前面的项

- **Delete 键**：清除选中行的自定义名称
  - 框选或选中一行或多行后，按 Delete 键可清除自定义名称
  - 清除后，该行将使用模式串计算的结果

## 使用技巧

1. **快速连续编辑**：双击第一行进入编辑，输入完成后按 Enter，会自动跳转到下一行继续编辑

2. **批量粘贴**：从 Excel 或其他应用复制多行文本，选中对应的行，按 Ctrl+V 即可批量填充

3. **取消编辑**：如果输入错误，按 ESC 键可以取消本次编辑，不会保存任何修改

4. **自定义名称优先级**：当自定义名称有值时，会覆盖模式串的计算结果；清除自定义名称后，会恢复使用模式串计算结果
